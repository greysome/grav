# grav: Simple gravity simulator app

## Building
```cargo build``` then ```cargo run```

## Keys
- **click**: select body
- **shift+click**: add body
- **left/right**: change speed
- **up/down**: change scale
- **p**: pause/unpause
- **r**: reverse/unreverse
